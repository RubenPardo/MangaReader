# MangaReader

## Descripción
Aplicación que consume la API de [MangaDex](https://api.mangadex.org/docs/).
- Buscador de Mangas
- Guardar Favoritos
- Leer capitulos
- Guardar paneles de manga


## Tecnologías y Arquitectura
- MVVM + Clean
- Koin
- Retrofit
- Room
